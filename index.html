Platform jumper

2D Browser-Based Jumping Game

Creator

Kevin Cordova

Application Overview

Platform Jumper is a simple 2D game where the player controls a character that jumps between platforms to reach higher levels. The goal is to achieve the highest possible score without falling off the platforms. This game demonstrates basic game mechanics, player control, and interactive physics in a browser environment.

Technology Overview

  • Languages: HTML, CSS, JavaScript
  • Libraries: HTML5 canvas 2D API
  • Platforms: Browser-based (Windows, MacOS, and Linux)
  • Hosting: GitHub Pages

Proposal Last Updated

December 7, 2025

Feature List (Prioritized)

  • Character movement using arrow keys (left and right)
  • Platforms that randomly generate as the player ascends
  • Collision detection to prevent falling through platforms

Future Feature List

  • Power-up (double jump)
  • Moving platforms
  • High score leaderboard

Basic Technical Features

  • HTML canvas for game display
  • JavaScript logic for movement, jumping, and platform generation
  • Simple collision detection

Advanced Technical Features

  • Smooth animations and physics calculations for realistic jumping
  • Screen shake effect when landing hard and during death
